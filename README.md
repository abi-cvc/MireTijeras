# MireTijeras - Sistema de Gestión de Citas, Reseñas y Sugerencias para Salón de Belleza
MireTijeras es una plataforma web para la gestión de citas, reseñas reales con autenticación Google, sugerencias de clientes y administración de convenios y beneficios para empresas. Pensada para salones de belleza que buscan digitalizar su atención y mejorar la experiencia de sus clientes.

URL: https://miretijeras.vercel.app
Backend: https://miretijeras.onrender.com

## Funcionalidades Principales
### Para clientes
Reserva de citas online con selección de servicio, fecha y hora.
Reseñas reales: Solo usuarios autenticados con Google pueden dejar reseñas, incluyendo foto, procedimiento y edad.
Buzón de sugerencias: Formulario anónimo para enviar sugerencias, con animación de sobre.
Consulta de beneficios y convenios: Información y formulario para empresas interesadas en convenios.

### Para administradores
Panel de administración: Visualización y gestión de citas, reseñas y sugerencias.
Login seguro: Acceso restringido con credenciales de administrador.
Pinear reseñas: Destacar reseñas importantes.
Eliminar reseñas o sugerencias: Moderación de contenido.

### Público general
Página de servicios: Catálogo de servicios ofrecidos.
Página de reseñas: Experiencias de clientes verificadas.
Página de convenios y beneficios: Información para empresas y grupos.
